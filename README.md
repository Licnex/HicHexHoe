# Hix Hex Hoe

Essentially Hexagonal tic tac toe by <a herf= https://www.youtube.com/@webgoatguy> webgoatguy </a>.

## Overview

- Two players: `X` and `O`
- `X` goes first and places **1** piece on the first turn
- After that, players alternate turns placing **2** pieces each turn
- First player to get **6 in a row** in any hex direction wins immediately

## How To Play

just go to the <a herf=https://licnex.github.io/HicHexHoe>HixHexHoe</a> website.

## Controls

- **Left click**: Place a piece on the hovered hex (if empty)
- **Drag (hold left mouse)**: Pan the board
- **Rules button**: Open game rules modal
- **Theme button**: Toggle light/dark mode
- **Reset button**: Start a new game
